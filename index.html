Python Implementation
import requests headers = { "Authorization": "Bearer YOUR_API_KEY", # Replace with your actual API key "Content-Type": "application/json" } payload = { "prompt": "A neon dragon flying through a futuristic cityscape", "duration": 15, "resolution": "1080p", "style": "cyber-dream", "audio_track": "synthwave" } response = requests.post("https://api.wisitrix.com/v1/nivi-on/generate", json=payload, headers=headers) print(response.json())

Python Implementation Guide:

  • Import the requests library for HTTP communication
  • Configure authentication headers with your API key
  • Define video generation parameters in payload
  • Send POST request to nivi-ON API endpoint
  • Process response containing generated video URL
JavaScript Implementation
fetch("https://api.wisitrix.com/v1/nivi-on/generate", { method: "POST", headers: { "Authorization": "Bearer YOUR_API_KEY", // Replace with your actual API key "Content-Type": "application/json" }, body: JSON.stringify({ prompt: "A neon dragon flying through a futuristic cityscape", duration: 15, resolution: "1080p", style: "cyber-dream", audio_track: "synthwave" }) }) .then(response => response.json()) .then(data => { console.log("Video URL:", data.video_url); }) .catch(error => { console.error("Error:", error); });

JavaScript Implementation Guide:

  • Use native fetch API for HTTP requests
  • Configure POST method with authentication headers
  • Stringify video parameters in request body
  • Handle response with promise chaining
  • Log video URL or catch and handle errors
